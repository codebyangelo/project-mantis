**What Happened:** The suspect, an employee in the technology development division, exfiltrated sensitive project files, specifically '[secret_project]_design_concept.ppt' and '[secret_project]_proposal.docx', to an unauthorized external storage device.

**Where Transferred:** Unauthorized removable media (USB drive identified as 'Authorized USB' on E: drive).

**How Stolen:** The suspect utilized LNK files to access and stage sensitive project data from the company's internal systems onto an unauthorized USB storage device, violating corporate security policies regarding removable media.

**When Occurred:** 2015-03-23 between 18:37:54 UTC and 18:38:23 UTC.

**Incident Narrative:**
On March 23, 2015, an internal investigation identified unauthorized data handling by an employee, referred to as 'Mr. Informant'. Forensic analysis of the workstation revealed the creation of two LNK files in the user's Recent items folder: '[secret_project]_proposal.LNK' at 18:37:54 UTC and '[secret_project]_design_concept.LNK' at 18:38:23 UTC. These files were linked to an external volume labeled 'Authorized USB' mapped to the E: drive, which contained directories for 'Secret Project Data'. The presence of these files confirms that the suspect accessed and staged sensitive project documentation onto an unauthorized removable storage device, a direct violation of the company's information security policy which prohibits the use of non-authorized portable storage devices. The activity is consistent with a deliberate attempt to exfiltrate proprietary technology-related intellectual property.

## 7. NIST SP 800-61r2 Alignment & Containment Recommendations
**Incident Lifecycle Phase:** Detection & Analysis -> Transitioning to Containment, Eradication, & Recovery

### Immediate Containment Actions
* Immediate revocation of the suspect's network and system access credentials.
* Physical seizure and isolation of the suspect's workstation and any associated removable media.
* Implementation of a temporary block on all USB mass storage device classes via Group Policy Object (GPO) across the affected subnet.

### Eradication & Recovery
* Perform a full forensic wipe and re-imaging of the compromised workstation.
* Purge all unauthorized LNK artifacts and temporary files associated with the exfiltration event from the user profile.
* Conduct a comprehensive audit of the 'Secret Project' file shares to identify any further unauthorized access or data staging.

### Post-Incident Activity
* Enhance endpoint telemetry to alert on the creation of LNK files pointing to removable media paths.
* Deploy Data Loss Prevention (DLP) solutions to monitor and block unauthorized file transfers to external storage devices.
* Conduct mandatory security awareness training focusing on the risks of unauthorized removable media and the company's data handling policies.
